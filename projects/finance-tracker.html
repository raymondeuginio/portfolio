Personal Financial Tracker

Next.js | PostgreSQL | TailwindCSS | Chart.js

Overview

This tracker helps young professionals manage expenses, monitor investments, and set savings targets. I built automated imports from bank statements and crafted a simple UI that surfaces insights without overwhelming graphs.

Tech Stack

Next.js TypeScript PostgreSQL Prisma TailwindCSS Chart.js

Features

  • Bank CSV importer with category suggestions powered by rules engine.
  • Interactive charts for cash flow, savings progress, and debt payoff.
  • Budget alerts delivered through email digests.
  • Goal timeline that visualizes milestones and celebrates wins.

Screenshots

Outcome

Freelance clients reported clearer budgeting decisions within the first month. Building the tracker deepened my TypeScript discipline and experience with server-side rendering patterns in Next.js.
