To-Do List Web App

React | Node.js | MongoDB | TailwindCSS

Overview

I built this web app to help students balance class assignments and personal tasks. The interface focuses on clarity, enabling drag-and-drop between lists, reminders, and a streak tracker that rewards consistent progress.

Tech Stack

React Node.js Express MongoDB TailwindCSS

Features

  • Kanban-inspired drag-and-drop between Today, Upcoming, and Done lists.
  • Recurring task scheduling with email reminders.
  • Focus mode with Pomodoro-style timer integration.
  • Progress analytics summarizing weekly completions and categories.

Screenshots

Outcome

The app is used by a small community of students who report fewer missed deadlines. Building it strengthened my React state management patterns and ability to ship fullstack experiences quickly.
